# UI/UX 고도화 및 시스템 테마 연동 작업

## 목표
- 앱의 시각적 안정성을 확보하고 현대적인 테마 설정 및 애니메이션 캐릭터 기능을 구현하여 완성도를 높임.

## 작업 항목
- [x] 사이드바 레이아웃 안정화 (상단 고정 `AlignTop` 적용 및 클릭 시 간격 변화 해결)
- [x] 메인 윈도우 크기 최적화 (`980x860` 확대) 및 설정 항목 시인성 확보
- [x] 테마 설정 시스템 개편 (라이트/다크/시스템 드롭다운 통합)
- [x] `QComboBox` 디자인 현대화 (애플/토스 스타일 미니멀 디자인 및 호버 텍스트 강조)
- [x] 오버레이 알림 캐릭터 이미지 확장 (`character/` 폴더 기반 GIF/PNG 지원)
- [x] 오버레이 크기 확대 (`620x340`) 및 캐릭터 이미지 대형화 (`200x200`)
- [x] 오버레이 닫기 버튼(X) 위치 모서리 최적화
- [x] 시스템 테마 실시간 변경 감지 및 반영 로직 구현
- [x] 라이트 모드 가독성 문제 해결 (`App Info` 색상 및 드롭다운 호버 대비 강화)

## 완료 기준
- 어떤 메뉴를 클릭해도 사이드바 간격이 일정하게 유지됨.
- 시스템 테마 변경 시 앱이 즉시 반응하여 테마를 전환함.
- `character/` 폴더에 이미지를 넣으면 알림 시 대형 캐릭터가 애니메이션과 함께 노출됨.
