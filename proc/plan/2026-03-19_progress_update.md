## 해결 기획 (Performance Strategy)
1. **백엔드 로그 제거**: `DEBUG: 현재 피로도 = ...` 등의 실시간 터미널 출력을 주석 처리하여 성능 확보.
2. **비동기 연결 보장**: React의 `useEffect` 내에서 브릿지 초기화 시 메인 렌더링을 방해하지 않도록 최적화.
3. **로컬 주소 표준화**: `127.0.0.1`로 모든 통신 포트를 통일하고, Vite 서버를 해당 주소로 고정.
4. **트레이 메뉴 고도화**: 메뉴 객체를 부모(App)와 연결하여 무조건 작동하게 보장.
