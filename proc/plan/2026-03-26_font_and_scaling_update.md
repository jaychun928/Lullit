# 폰트 가독성 상향 및 저해상도 렌더링 최적화 (v0.3.1)

## 목표
FHD(1080p) 해상도에서의 폰트 깨짐 현상을 해결하고, 전역 폰트 크기를 +1px 상향하여 가독성을 확보함. 폰트 크기 증가에 따른 텍스트 잘림 현상을 방지하기 위해 UI 레이아웃을 보정함.

## 작업 항목
- [x] FHD 해상도 스케일링 로직 복구 (최소 0.6배율 유지)
- [x] 전역 폰트 렌더링 최적화 (AA 활성화, PreferFullHinting 적용)
- [x] `style.qss` 내 모든 폰트 크기 일괄 상향 (+1px)
- [x] 텍스트 잘림 방지를 위한 UI 요소 보정 (PrimaryBtn, CaptionText 등 min-height 상향)
- [x] 변경된 디자인 및 기술 명세 문서 업데이트 (`proc/spec/`)
