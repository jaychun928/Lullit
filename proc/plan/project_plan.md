# 작업 계획서 (Project Plan) - "Lullit" v0.4.0 Beta

## 목표
- [x] Toss/Apple 스타일의 미니멀리즘 다크 모드 UI 구현
- [x] 사용자 활동 기반 피로도 점수 엔진 및 최적화 완료
- [x] 유령 오버레이(Click-through) 및 랜덤 스트레칭 연동 완료
- [x] 베타 배포를 위한 EXE 패키징 및 최종 검수 (v0.4.0)

## 작업 항목
### Phase 1: 기반 구축 (완료)
- [x] `uv` 기반 Python 개발 환경 구축
- [x] PySide6 기반 네이티브 윈도우 UI 구조 설계
- [x] Pretendard 폰트 및 다크 모드 QSS 테마 적용

### Phase 2: 핵심 엔진 및 로직 (완료)
- [x] 전역 입력 리스너 및 피로도 가산 알고리즘 구현
- [x] CPU 점유율 최적화 (1.0초 샘플링 방식)
- [x] 휴식 감지 및 자동 점수 회복 로직 구현
- [x] 소스 내장형(`stretches_data.py`) 데이터 관리 체계 구축

### Phase 3: UI/UX 고도화 (완료)
- [x] 사이드바 IA 설계 및 페이지 전환 스택 구현
- [x] 홈 대시보드 및 진행률 시각화(Shadow 효과)
- [x] 설정 메뉴별 명시적 '저장' 버튼 및 Toast 알림 연동
- [x] 닫기 버튼 접근성 개선 및 위치 최적화 (우측 70px, 상단 20px)

### Phase 4: 알림 오버레이 (완료)
- [x] Win32 API 연동 유령 오버레이(Ghost Overlay) 구현
- [x] Fade 인/아웃 애니메이션 및 자동 숨김 타이머 (20초)
- [x] 오버레이 내부 랜덤 스트레칭 텍스트 표시 로직 (잘림 방지 최적화)

### Phase 5: 최종 안정화 및 배포 (진행 중)

    - [x] 기준 해상도 상향 조정 (`2560x1440 QHD` 기준 100% 스케일링)
    - [x] `main_window.py` 및 `overlay.py` 모든 픽셀(px) 수치 -> `dp()` 동적 보정 완료
    - [x] QSS 내 폰트 및 여백(px) 값 실시간 치환 로직(`scale_qss`) 적용
- [x] 폰트 가독성 상향 및 저해상도(FHD) 최적화 (v0.3.1)
    - [x] 전역 폰트 크기 +1px 상향 및 버튼/캡션 영역 잘림 방지 레이아웃 보정

    - [x] 2열 그리드 목록 ↔ 상세 단계 리스트 간 매끄러운 스택 전환 구현
- [x] 네비게이션 사용성 개선 및 메모리 최적화 (v0.4.0)
    - [x] 메뉴 이동 시 각 페이지 상태 초기화 및 저장 안 된 설정 자동 폐기 로직 구현
    - [x] 상세 페이지 이탈 시 위젯 즉시 파괴(`deleteLater`) 및 `gc.collect()` 호출 강화
