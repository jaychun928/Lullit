# 01. AI 명령지침 (Core Rules / System Prompt)

- 웹뷰 기반의 복잡한 프론트엔드/백엔드 브릿지를 폐기하고, 순수 본연의 PySide6 데스크톱 앱으로 전면 재작성한다.
- 렉과 연결 오류의 주원인이었던 `pywebview` 구조를 제거하고 빠르고 가벼운 네이티브 UI로 구성한다.
- `input/`의 요구사항에 명시된 토스/애플 스타일의 미니멀리즘과 QFrame 기반의 레이아웃을 충실히 따른다.
- 오버레이 알림은 포커스를 뺏지 않아야 하며 (Click-through), Win32 API 및 Qt 특성을 사용해 유령 창으로 구현한다.
- 키보드/마우스 트래킹 시 무한 루프가 아닌 샘플링 단위로 처리하여 CPU 낭비를 막는다.
