# 05. 운영 로직 및 비즈니스 정책 (Business Rules / Validation)

### 피로도 측정 규칙
- `TIME_UNIT_SCORE`: 1 (1초당 1점)
- `MOUSE_CLICK_SCORE`: 0.5 (클릭 1회당 0.5점)
- `MOUSE_MOVE_SCORE` / `MOUSE_MOVE_UNIT`: 1000픽셀 이동마다 1점
- `KEY_INPUT_SCORE`: 0.2 (키보드 1타당 0.2점)

### 알림 타이밍
- `FATIGUE_THRESHOLD`: 기본값은 3600(60분)이나, 사용자가 **[알림 상세 설정]** 메뉴에서 지정한 분(Minute) 단위에 따라 동적으로 할당 및 변동됩니다. 점수 도달 시 알림 오버레이를 띄우고 홈 화면의 카운트를 1 증가시키며 점수를 0으로 리셋합니다.

### 복구 (Recovery) 규칙
- `IDLE_DETECTION_TIME`: 300초 입력 없을 시 휴식 진입
- `IDLE_RECOVERY_RATE`: 0.1 (1분마다 전체 점수의 10% 차감)
- 알림이 뜨는 즉시 피로도 점수를 0으로 완전 초기화 (BREAK_VALIDATION 로직 삭제).
