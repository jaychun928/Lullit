# 07. 브랜딩 및 디자인 (Logo / CI / Typography / Color Palette)

### 1. 브랜드 컨셉 (Brand Mood)
- **Toss/Apple 미니멀리즘**: 불필요한 장식을 배제하고 여백과 깔끔한 타이포그래피로 신뢰감과 편안함을 전달.
- **반투명 효과 (Glassmorphism)**: 윈도우 배경이 살짝 비치는 효과를 사용하여 현대적이고 고급스러운 느낌 구현.

### 2. 컬러 팔레트 (Color Palette)
- **Main Color**: #33BA25 (Lullit Deep Green) - 생기 있는 건강함을 상징.
- **Background**: 
  - Dark: #121212 (메인), #1C1C1E (카드/사이드바)
  - Light: #F2F2F7 (메인), #FFFFFF (카드/사이드바)
- **Text**: 
  - Main: #FFFFFF (Dark) / #1C1C1E (Light)
  - Sub: #8E8E93 (Dark) / #3C3C43 (Light)
  - Translucent: rgba(255, 255, 255, 0.4) (Dark) / rgba(0, 0, 0, 0.4) (Light) - 보조 메뉴용
  - Light Mode Default: #AEAEB2 (드롭다운 비활성/기본 상태)

### 3. 아이콘 시스템 (Iconography)
- **다크 모드**: `theme_dark` 폴더의 **흰색/밝은 아이콘** 사용.
- **라이트 모드**: `theme_light` 폴더의 **검정/어두운 아이콘** 사용.
- **아이콘 교체**: 테마 변경 저장 시 모든 메뉴 아이콘이 즉시 동기화됨. (SVG 기반)

### 4. 주요 UI 컴포넌트 디자인
- **Primary Button**: #33BA25 배경, 흰색 텍스트, 12px 둥근 모서리.
  - **가독성**: 폰트 크기 19px, 최소 높이(min-height) 48px 적용으로 텍스트 잘림 방지.
- **Menu Button**: 19px 크기. 활성화 시 Lullit Green 배경 적용. 사이드바 상단 고정 레이아웃.
- **Feedback & Exit Button**: 15px 크기, 메뉴 버튼과 동일한 여백(Padding/Margin) 적용으로 간격 통일.
- **QComboBox (드롭다운)**: 12px 둥근 모서리, 미세한 경계선 적용.
  - **폰트**: 17px 크기로 가독성 확보.
  - **호버 방식**: 배경색 변화 대신 **텍스트 색상 강조** 방식 채택.
- **Caption & Body Text**: 
  - **Body**: 18px, 행간 1.6 설정.
  - **Caption**: 19px, 최소 높이(min-height) 55px 상향 조정하여 긴 문장 출력 시 잘림 방지.
- **Window Close Button**: 우측 상단 ✕ (WindowCloseBtn), 23px 크기, 테마 대응(Dark 40% White / Light 40% Black).
