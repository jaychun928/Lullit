# 06. 콘텐츠 데이터셋 (Seed Data / Content Samples)

- 스트레칭 알림 동작 예시 (`input/스트레칭 정리.txt` 반영):
  1. **손목 스트레칭 1**: 손목 방향 상하 지긋이 누르기 (각각 5초)
  2. **손목 스트레칭 2**: 주먹 강하게 쥐었다 펴기 (5초)
  3. **뒷목 스트레칭**: 고개 젖히기 (10초)
  4. **어깨 스트레칭**: 어깨 위 손 올리고 돌리기 (각 5회)
  5. **허리 스트레칭**: 깍지 끼고 좌우 굽히기 (5초)
  6. **눈 마사지**: 눈 돌리기 및 누르기 (5바퀴)

상기 텍스트 데이터는 Json이나 딕셔너리로 관리하여 유지보수성을 확보한다.
