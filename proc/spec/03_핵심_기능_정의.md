# 03. 핵심 기능 정의 (FSD / IA / Sitemap)

### MoSCoW
- **Must Have**: 
  - 피로도 가중치 알고리즘(마우스/키보드/시간 트래킹).
  - 유령 오버레이(포커스 탈취 없는, 클릭이 관통하는 반투명 알림).
  - 메인 대시보드 (퍼센트 바, 퀵 액션).
- **Should Have**: 
  - 미니멀리즘 디자인 시스템(Pretendard 폰트, 둥근 모서리, 그림자).
  - 기본 알림 주기 설정 (스핀박스, 5~120분 자유 설정).
  - 테마 설정 (라이트/다크/시스템 설정 동기화).
  - 애니메이션 알림 (GIF 캐릭터 및 이미지 지원).
- **Could Have**: 
  - 스트레칭 클래스 콘텐츠, 상세 캐릭터 선택 UI.
- **Won't Have**: 
  - 현재 버전에서의 마이페이지/온라인 샵 연동 로그인 시스템 등.

### Sitemap & IA
1. **홈 (Home)**
   - 퀵 액션 기능
   - 오늘 요약 (알림 뜬 횟수)
   - 퍼센트 바
   - 이용 팁 확인하기 (팝업)
2. **스트레칭 클래스 (개편)**
   - **카테고리**: 목, 어깨, 허리, 손목, 눈 등 5개 부위 라이브러리.
   - **상세 정보**: 각 부위 클릭 시 상세 동작 리스트 및 방법 확인 가능.
   - **UI**: 2열 그리드 목록 및 내부 상세 스택 전환 구조.
3. **추천 샵 (준비 중)**
4. **알림 상세 설정**
   - 알림 주기 설정 (QSpinBox, 5~120분)
   - 오버레이 투명도 설정 (QSlider, 20~100%)
   - [자동 폐기]: 저장하지 않고 메뉴 이동 시 변경사항 초기화.
5. **고급 설정**
   - 시작 프로그램 등록 여부
   - 테마 모드 선택 (라이트/다크/시스템 동기화 드롭다운)
   - 앱 정보 및 버전 확인
   - 법적 고지 (이용약관, 개인정보 처리방침)
6. **마이페이지 (준비 중)**
