# 09. 단계별 개발 로드맵 (WBS / Milestones)

- **Phase 1: 기획 및 아키텍처 재설정**
  - 명세서 작성 및 레거시(WebView/React) 파일 완전 제거. 승인 요청.
- **Phase 2: 핵심 알림 및 로직 구축**
  - FatigueEngine 재구축 (샘플링 최적화, CPU 점유율 하락).
  - 유령 오버레이 알림 UI (Click-Through) 테스트 완료.
- **Phase 3: 메인 UI 구현 및 미니멀리즘 확정**
  - 토스/애플 스타일 기반(Pretendard 폰트, #d5e6e6 배경, 투명 오버레이) 메인 UI 적용 및 트레이 아이콘 버그 완전 수정.
- **Phase 4: 코어 기능 연동 및 설정 적용 (완료 단계)**
  - ConfigManager를 이용한 알림 주기(분) 동기화 구동 확정.
  - winreg 레지스트리를 통한 시작 프로그램(자동 실행) 등록 기능 완성.
- **Phase 5: 배포 준비 및 최적화 (향후 스텝)**
  - PyInstaller를 사용한 `.exe` 배포 빌드 및 실행 콘솔 윈도우(터미널) 숨김 작업.
