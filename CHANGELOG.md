# Changelog

모든 주요 변경 사항이 이 파일에 기록됩니다.

## [0.4.0] - 2026-04-06
### Added
- **베타 정식 배포 준비**: v0.4.0으로 버전을 격상하고 전체 시스템 안정성 검수를 완료했습니다.
- **스트레칭 클래스 콘텐츠 고도화**: 부위별 상세 스트레칭 데이터를 보강하고 UI 레이아웃을 최적화했습니다.

### Changed
- **네비게이션 로직 개선**: 페이지 전환 시 메모리 해제 및 상태 초기화 로직을 강화하여 장시간 실행 시의 안정성을 높였습니다.
- **버전 표기 업데이트**: 앱 내 정보 창 및 시작 로그의 버전 표기를 0.4.0으로 일괄 갱신했습니다.

### Optimized
- **빌드 프로세스 최적화**: PyInstaller spec 파일 및 Inno Setup 스크립트를 최신화하여 설치 프로그램 배포 준비를 마쳤습니다.

## [0.3.1] - 2026-03-26
### Fixed
- **해상도 독립적 스케일링 최적화**: 기준 해상도(2560x1440) 대비 논리적 해상도 비율 계산 로직을 단순화하여 1080p 등 저해상도에서의 레이아웃 깨짐 현상을 근본적으로 해결했습니다.
- **메인 윈도우 닫기 버튼(X) 위치 보정**: `resizeEvent`를 통해 창 크기 변경 시에도 닫기 버튼이 항상 우측 상단 최적의 위치에 머물도록 동적 배치를 적용했습니다.
- **레이아웃 누락 스케일링 적용**: 사이드바 너비, 팝업 창 크기, 각종 여백(Margin) 및 간격(Spacing)에 `dp()` 처리가 누락되었던 부분들을 모두 보완하여 해상도 대응력을 높였습니다.

## [0.3.0] - 2026-03-26
### Added
- **전역 해상도 대응 시스템 (Dynamic Scaling)**: QHD(2560x1440)를 기준으로 모든 해상도(FHD 등)에서 UI 크기가 자동으로 조절되도록 설계했습니다.
- **스트레칭 클래스 전면 개편**: 부위별(목, 어깨, 허리, 손목, 눈) 카테고리 도입 및 상세 보기 내비게이션 기능을 추가했습니다.
- **DPI Awareness 적용**: 윈도우 OS의 배율 설정에 따라 텍스트가 흐릿해지지 않고 선명하게 보이도록 고해상도 대응을 완료했습니다.

### Changed
- **메뉴 상태 초기화 로직**: 사이드바 메뉴 클릭 시 항상 페이지 초기 상태(목록)를 보여주어 사용자 혼란을 방지합니다.
- **설정 자동 동기화**: 저장을 누르지 않고 메뉴를 이동할 경우 변경사항이 자동 폐기되어 설정 오류를 방지합니다.
- **알림 레이아웃 고정**: 캐릭터 및 메시지 텍스트의 위치를 틀 구조로 고정하여 시각적 안정성을 높였습니다.

### Optimized
- **상세 뷰 메모리 관리**: 스트레칭 상세 보기 이탈 시 위젯을 즉시 파괴(`deleteLater`)하여 상주 메모리를 최소화했습니다.
- **가비지 컬렉션 강화**: 페이지 전환 및 테마 변경 등 주요 이벤트 발생 시 즉시 메모리 정리를 수행합니다.

## [0.2.8] - 2026-03-25
### Optimized
- **메모리 점유율 획기적 개선**: 앱을 오래 켜두어도 메모리 사용량이 늘어나지 않도록 리소스 관리 로직을 전면 재설계했습니다.
- **이미지 리소스 최적화**: 트레이 아이콘 및 캐릭터 이미지 로딩 시 불필요한 메모리 할당을 방지하고, 사용 후 즉시 해제하도록 개선했습니다.
- **아이콘 캐싱 시스템**: 테마(다크/라이트) 변경 시 아이콘 객체를 재사용하여 객체 생성 오버헤드를 줄였습니다.
- **오버레이 성능 향상**: 알림 오버레이가 보이지 않을 때 GIF 애니메이션을 완전히 정지시켜 백그라운드 CPU/메모리 소모를 차단했습니다.
- **UI 렌더링 최적화**: 스타일시트 적용 방식을 개선하여 테마 전환 속도를 높이고 불필요한 위젯 재그리기 연산을 줄였습니다.
- **가비지 컬렉션(GC) 튜닝**: 파이썬의 세대별 가비지 컬렉션 임계값을 앱 특성에 맞춰 조정하여 메모리 파편화를 방지했습니다.

## [0.2.6] - 2026-03-25
### Fixed
- **타 컴퓨터 실행 호환성 개선**: 특정 환경에서 UI와 트레이 아이콘이 나타나지 않고 백그라운드 프로세스만 남던 현상 해결.
- **지연된 엔진 시작**: 시스템 후킹(pynput) 초기화 로직을 UI 표시 이후로 분리하여 앱 시작 시 프리징 현상 방지.
- **싱글 인스턴스 안정화**: 비정상 종료 후 잔류하는 소켓 서버를 강제로 정리하여 중복 실행 방지 로직의 신뢰도 향상.
- **DPI 및 그래픽 최적화**: 고해상도 모니터 지원(High DPI Scaling) 설정을 추가하고 소프트웨어 렌더링 옵션을 강화.
- **빌드 종속성 보강**: PyInstaller 빌드 시 pynput 관련 Windows 백엔드 모듈이 누락되지 않도록 명시적으로 포함.
